### 1. 前端开发

涵盖 HTML、CSS、JavaScript、TypeScript、浏览器与网络、React/Vue 等框架、工程化与性能优化、前端架构与监控，以及 AI Coding（架构、Rules、Skills 等实践笔记）。

### 2. AI 技术

生成式 AI 导论（LLM 训练、提示工程、Agent、Transformer 等），以及 LangChain 应用开发与 Agent 相关笔记。

### 3. 工具使用

### 4. 后端开发

以 Python 为主：语言基础、标准库、网络编程、与数据库（如 MySQL）的结合等。
