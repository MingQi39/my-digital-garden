# 双机热备 HA

- PostgreSQL+RHCS
	- RHCS 是一个高可用性、高可靠性、负载均衡、存储共享且经济廉价的集群工具集合。
- PostgreSQL+Pacemaker+Corosync
	- Pacemaker 是一个集群资源管理器。它利用集群基础构件(corosync)提供的消息和成员管理能力来探测并从节点或资源级别的故障中恢复，以实现群集服务(亦称资源)的最大可用性

# 主从复制

- PostgreSQL 流复制+keepalived
	- 成本低、使用简单、有数据一致性的风险切换后可能要重搭备库
- PostgreSQL 流复制+repmgr 
	- 安装简单、功能强大、知名公司做后盾
- PostgreSQL流复制+pgpool
	- 读写分离、读的负载均衡、连接池
	- 配合脚本可以自动failover、配置较复杂BUG较多，相关功能还在完善中。
