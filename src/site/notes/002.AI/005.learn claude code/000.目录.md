# 背景

2026 年 3 月 31 日，安全研究员 [Chaofan Shou](https://x.com/Fried_rice) 在 X 上发了一条[推文](https://x.com/Fried_rice/status/2038894956459290963?s=20)：

> *"Claude Code source code has been leaked via a map file in their npm registry!"*

Anthropic 发布到 npm 的 Claude Code 包（`@anthropic-ai/claude-code`）里，附带了一个 `.map` 文件。这个 source map 指向了 Anthropic R2 存储桶中未混淆的 TypeScript 源码

整个 `src/` 目录就这样暴露在了公网上，包含 **1902 个源文件，512,685 行 TypeScript**

```tree
npm包结构（异常版本）
├── cli.js          ← 正常混淆产物
├── cli.js.map      ← 59.8MB，灾难所在
└── package.json
```

相关仓库：

- [claw-code](https://github.com/ultraworkers/claw-code)：Rust 重构版
- [cc-haha](https://github.com/xurenda/cc-haha)：修复的本地可运行版本
- [free-code](https://github.com/paoloanzn/free-code)：剔除遥测数据、移除安全提示护栏、实验性功能解锁

**免责声明**：这是泄露的编译产物经 Source Map 还原的结果，非官方开源，仅供技术研究。所有原始源码版权归 [Anthropic](https://www.anthropic.com/) 所有
